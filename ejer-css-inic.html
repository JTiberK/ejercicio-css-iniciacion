Ejercicio 26: Efecto overlay

Enunciado:
Crea un efecto de superposición (overlay) sobre una imagen que se muestre al hacer hover. Combina técnicas de posicionamiento absoluto y transparencia (opacity) para lograr este efecto.

Texto de Superposición
        <div class="imagen-container">
            <img src="https://planetasmagicos.com/wp-content/uploads/2017/02/p.peque%C3%B1o-3-mod.jpg" alt="Imagen" class="imagen">
            <div class="overlay">Texto de Superposición</div>
        </div>
    
        <style>
     
        .imagen-container {
            position: relative; /* Necesario para posicionar el overlay */
            width: 200px; /* Ajusta el tamaño de la imagen */
        }
        
        .imagen {
            width: 100%; /* La imagen ocupa todo el contenedor */
            height: auto; /* Mantiene la relación de aspecto */
            display: block; /* Elimina el espacio debajo de la imagen */
        }
        
        .overlay {
            position: absolute; /* Posiciona el overlay sobre la imagen */
            top: 0;
            left: 0;
            right: 0;
            bottom: 0;
            background-color: rgba(0, 0, 0, 0.5); /* Fondo negro con opacidad */
            color: white; /* Color del texto */
            display: flex; /* Usamos flexbox para centrar el texto */
            justify-content: center; /* Centra horizontalmente */
            align-items: center; /* Centra verticalmente */
            opacity: 0; /* Inicialmente oculto */
            transition: opacity 0.3s ease; /* Transición suave */
        }
        
        .imagen-container:hover .overlay {
            opacity: 1; /* Muestra el overlay al hacer hover */
        }
        </style>

Ejercicio 27: Animación de carga

Enunciado:
Crea un spinner giratorio utilizando una animación circular con @keyframes y transform: rotate(...). Aprende a implementar indicadores visuales de carga. 

    <div class="spinner"></div>

    <style>

    .spinner {
        width: 100px; /* Ancho del spinner */
        height: 100px; /* Alto del spinner */
        border: 10px solid #3498db; /* Color y grosor del borde */
        border-top: 10px solid transparent; /* Color del borde superior */
        border-radius: 50%; /* Hace que el div sea circular */
        animation: spin 1s linear infinite; /* Aplicar la animación */
    }
    
    @keyframes spin {
        from {
            transform: rotate(0deg); /* Comienza en 0 grados */
        }
        to {
            transform: rotate(360deg); /* Termina en 360 grados */
        }
    }

    </style>
