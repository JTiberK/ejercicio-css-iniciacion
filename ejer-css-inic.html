Ejercicio 1: Cambiar color de texto (Inline CSS)

Enunciado:
Utiliza CSS inline para cambiar el color del texto de un párrafo a rojo. Aprende cómo aplicar estilos directamente en el HTML usando el atributo style. 

<p style="color: red;">Este es un párrafo con texto en color rojo utilizando CSS inline.</p>

Este es un párrafo con texto en color rojo utilizando CSS inline.

Ejercicio 2: Estilos internos (Tag <style>)

Enunciado:
Usa la etiqueta <style> dentro del archivo HTML para definir un estilo que cambie el color de todos los títulos <h1> a azul. Practica cómo aplicar estilos internos al documento. 

            <style>
                h1 {
                    color: blue;
                }
            </style>

Ejercicio 3: Enlace a CSS externo

Enunciado:
Crea un archivo externo llamado styles.css y vincúlalo a tu archivo HTML mediante la etiqueta <link>. Define un estilo básico en este archivo para entender cómo organizar tus estilos en archivos separados. 

<link rel="stylesheet" href="styles.css">

Ejercicio 4: Selectores de clase

Enunciado:
Crea una clase .destacado en tu hoja de estilos que ponga el texto en negrita (font-weight: bold). Aplica esta clase a un elemento en tu HTML para ver su efecto. 

        
            .destacado {
                font-weight: bold;
            }

Ejercicio 5: Selectores de ID

Enunciado:
Usa un selector de ID (#contenedor) para centrar un <div> horizontalmente. Configura su ancho y utiliza margin: 0 auto; para lograrlo. 

        
            #contenedor {
                width: 600px;
                margin: 0 auto;
            }

Ejercicio 6: Agrupar selectores

Enunciado:
Aplica el mismo estilo a los elementos <h2> y <h3> agrupando sus selectores en una sola regla. Cambia su familia de fuentes y color para practicar la reutilización de estilos. 

        
            h2,
            h3 {
                font-family: 'Arial', sans-serif;
                color: rgb(24, 128, 197);
            }

Enunciado:
Centra un <div> de 600px de ancho horizontalmente utilizando margin: 0 auto;. Aprende cómo el uso de márgenes automáticos puede simplificar el centrado de elementos. 

<div style="width: 600px; margin: 0 auto;">Este es un div centrado horizontalmente.</div>

Ejercicio 10: Fuentes personalizadas

Enunciado:
Incluye una fuente personalizada de Google Fonts en tu proyecto y aplica esta fuente al cuerpo del documento (<body>). Aprende a mejorar la apariencia visual de tu sitio web. 

<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400&display=swap" rel="stylesheet"></link>
    <style>
        body {
            font-family: 'Roboto', sans-serif;
        }
    </style>

Ejercicio 11: Fondo degradado

Enunciado:
Crea un fondo degradado que pase de azul a verde utilizando la propiedad background: linear-gradient(...). Explora cómo los gradientes pueden agregar profundidad a tus diseños.

Ejercicio 12: Border-radius

Enunciado:
Convierte un <div> cuadrado de 100x100px en un círculo perfecto usando border-radius: 50%;. Descubre cómo redondear bordes para crear formas más interesantes.

Ejercicio 13: Display inline-block

Enunciado:
Muestra tres <div> en línea conservando su ancho específico mediante display: inline-block;. Aprende a controlar el flujo de elementos en una página.

Ejercicio 14: Posicionamiento relativo

Enunciado:
Mueve un elemento 15px hacia abajo desde su posición original usando position: relative; y la propiedad top. Familiarízate con el posicionamiento relativo en CSS.

Ejercicio 15: Posicionamiento absoluto

Enunciado:
Coloca un botón en la esquina superior derecha de su contenedor utilizando position: absolute; junto con las propiedades top y right. Aprende cómo el posicionamiento absoluto permite colocar elementos con precisión.

Ejercicio 16: Menú con Flexbox

Enunciado:
Crea un menú horizontal centrado utilizando Flexbox. Usa propiedades como display: flex;, justify-content: center; y gap para distribuir los elementos uniformemente.

Ejercicio 17: Transición simple

Enunciado:
Haz que un botón cambie de color suavemente al pasar el mouse sobre él utilizando la propiedad transition. Descubre cómo las transiciones añaden interactividad visual.

Ejercicio 18: Animación con keyframes

Enunciado:
Crea una animación que desplace un elemento de izquierda a derecha utilizando @keyframes. Aprende cómo las animaciones pueden dar vida a tus diseños.

Ejercicio 19: Media Queries

Enunciado:
Oculta un elemento en pantallas menores de 600px de ancho usando una media query con @media (max-width: 600px). Practica cómo adaptar tus diseños a diferentes dispositivos.

Ejercicio 20: Pseudo-elementos (::before)

Enunciado:
Agrega un ícono antes de cada enlace (<a>) utilizando el pseudo-elemento ::before y la propiedad content. Aprende a enriquecer tus diseños con contenido generado por CSS.

Ejercicio 21: CSS Variables

Enunciado:
Define una variable CSS (--color-primario) para almacenar un color primario y úsala en varios elementos. Aprende cómo las variables facilitan la gestión de estilos.

Ejercicio 22: Transformaciones 2D

Enunciado:
Rota una imagen 45 grados al hacer hover utilizando la propiedad transform: rotate(45deg);. Explora cómo las transformaciones permiten modificar visualmente los elementos.

Ejercicio 23: Box-shadow

Enunciado:
Añade una sombra suave a una tarjeta (<div>) usando la propiedad box-shadow. Mejora la percepción de profundidad en tus diseños.

Ejercicio 24: Texto en columnas

Enunciado:
Divide un párrafo largo en dos columnas utilizando las propiedades column-count y column-gap. Aprende cómo estructurar texto en múltiples columnas.

Ejercicio 25: Filtros CSS

Enunciado:
Convierte una imagen a escala de grises utilizando el filtro filter: grayscale(100%);. Descubre cómo los filtros pueden alterar la apariencia visual de los elementos.

Ejercicio 26: Efecto overlay

Enunciado:
Crea un efecto de superposición (overlay) sobre una imagen que se muestre al hacer hover. Combina técnicas de posicionamiento absoluto y transparencia (opacity) para lograr este efecto.

Ejercicio 27: Animación de carga

Enunciado:
Crea un spinner giratorio utilizando una animación circular con @keyframes y transform: rotate(...). Aprende a implementar indicadores visuales de carga.
